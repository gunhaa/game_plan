# Chat, Please

> 버튜버 방송의 채팅 모더레이터가 되어 실제 채팅 로그를 심사하는 게임

| 항목 | 내용 |
|------|------|
| 장르 | 퍼즐 / 시뮬레이션 / 인디 |
| 레퍼런스 | Papers, Please |
| 상태 | 기획 중 |

## 핵심 컨셉

Papers, Please의 서류 심사 구조를 채팅 검열에 대입한다.  
플레이어는 버튜버 소속사의 채팅 모더레이터로 고용되어, 실시간으로 올라오는 채팅을 보고 허용/삭제/밴을 판단한다.  
채팅 데이터는 실제 인터넷 방송 로그를 기반으로 가공해 사용한다.

## 폴더 구조

```
chat_please/
├── comparisons/
│   └── papers_please/        # 핵심 레퍼런스 분석
├── design/
│   ├── gdd.md                # 메인 기획 문서
│   ├── mechanics/            # 메카닉 상세
│   ├── systems/              # 시스템 설계
│   ├── narrative/            # 세계관 & 스토리
│   └── content/              # 콘텐츠 (채팅 DB, 버튜버, 규칙셋)
└── archive/                  # 보류 아이디어
```
